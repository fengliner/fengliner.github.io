初识Markdown

  • markdown
learning

为了能够写出漂亮的博客，学习到了markdown，然后一直在用，很喜欢这种写作方式。即简洁又漂亮，不仅方便写作者书写，也方便阅读者阅读。简书就是利用了markdown的这种优点。

本质上，类似于html，markdown也是一种标记语言，ie,用某些字符标示某种格式，只是标记格式不同罢了。

对于其它关于markdown的介绍，包括语法，编辑器等，可在用Markdown来书写你的博客里进行快速学习。

现在，markdown算是比较流行，而且简单易学，性价比很高！

注：个人现在用的文本编辑器是EverEdit+Markdown Reader，Markdown Reader是chrome插件，支持同步更新，强烈推荐。

使用方法：用EverEdit打开文本文件进行编辑，支持高亮显示，在“查看”菜单下选择“外部浏览器预览”即可在chrome中看到EverEdit中正在编辑的文本，非常方便。

好了，Markdown就介绍到这里，开始你的markdown之旅吧！
